# Cloud Music Player - 本地音乐播放器

一个精美的本地音乐播放器桌面应用，采用网易云音乐风格设计，支持桌面歌词显示和多主题切换。

## 功能特性

### 核心功能
- 🎵 **本地音乐播放** - 支持 MP3, FLAC, WAV, M4A, OGG, WMA, AAC 格式
- 📁 **文件夹扫描** - 自动扫描文件夹导入音乐
- 🔍 **智能搜索** - 按歌曲名、歌手、专辑搜索
- 📝 **歌词自动匹配** - 自动查找同文件夹下的同名歌词文件
- 🔗 **歌词持久绑定** - 每个音频只需绑定一次，永久记住
- 🖥️ **桌面悬浮歌词** - 支持拖拽移动、锁定/解锁
- 🎨 **多主题切换** - 5种精美主题：深邃黑、网易云红、海洋蓝、梦幻紫、森林绿
- 🎛️ **播放模式** - 列表循环/随机播放/单曲循环

### 界面特性
- 网易云音乐风格界面
- 自定义标题栏
- 播放列表管理
- 进度条拖拽控制
- 音量调节
- 歌曲封面显示
- 歌词绑定状态显示

## 使用说明

### 添加音乐
1. 点击"添加音乐"按钮选择音乐文件
2. 或点击"扫描文件夹"批量导入文件夹中的音乐

### 播放控制
- **播放/暂停**: 点击底部播放按钮、双击歌曲，或按空格键
- **上一首/下一首**: 点击底部控制按钮，或按 Ctrl+←/Ctrl+→
- **进度调节**: 拖动进度条
- **音量调节**: 拖动音量滑块

### 歌词功能

#### 自动匹配
播放歌曲时，播放器会自动查找同文件夹下的同名歌词文件：
- `song.mp3` → `song.lrc` 或 `song.txt`
- 自动匹配成功后，歌词绑定会永久保存

#### 手动加载歌词
1. 播放歌曲时，点击底部控制栏的歌词按钮，或按 L 键
2. 在弹出的歌词设置窗口中：
   - 点击"自动查找歌词"尝试自动匹配
   - 或点击"手动加载歌词"选择歌词文件
3. 点击"显示桌面歌词"开启桌面歌词窗口

#### 桌面歌词窗口操作
- **移动**: 解锁状态下，拖动顶部拖拽区域
- **锁定/解锁**: 点击锁定按钮，或按 L 键，或双击歌词区域
- **切换主题**: 点击主题按钮，或按 T 键，或右键点击歌词区域
- **关闭**: 点击关闭按钮，或按 ESC 键

### 主题切换
- 点击标题栏的主题按钮循环切换主题
- 或在侧边栏点击具体主题
- 5种主题可选：深邃黑、网易云红、海洋蓝、梦幻紫、森林绿

## 歌词文件格式

支持标准 LRC 格式歌词文件：

```
[00:00.00]歌曲标题
[00:02.50]歌手名
[00:05.00]第一句歌词
[00:08.50]第二句歌词
...
```

歌词绑定规则：
1. 播放歌曲时，自动查找同文件夹下的同名 `.lrc` 或 `.txt` 文件
2. 如果找到，自动加载并保存绑定关系
3. 绑定关系保存在应用数据目录，永久有效
4. 可以手动更改歌词绑定

## 快捷键

### 主窗口
| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `Ctrl + ←` | 上一首 |
| `Ctrl + →` | 下一首 |
| `L` | 打开歌词设置 |

### 桌面歌词窗口
| 快捷键 | 功能 |
|--------|------|
| `L` | 锁定/解锁窗口 |
| `T` | 切换歌词主题色 |
| `ESC` | 关闭歌词窗口 |
| `右键` | 切换主题色 |
| `双击` | 锁定/解锁窗口 |

## 数据存储

- **播放列表**: 保存在 `localStorage`
- **歌词绑定**: 保存在应用数据目录 (`userData/lyric-bindings.json`)
- **主题设置**: 保存在 `localStorage`

## 注意事项

1. 首次运行需要安装依赖，请确保网络连接正常
2. 桌面歌词窗口需要在播放歌曲时才会显示歌词
3. 歌词文件需要与音乐文件时间戳匹配才能正确同步
4. 歌词绑定是永久的，除非手动清除
